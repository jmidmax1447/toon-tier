# Toon Tier

Toon Tier is a web application where users can track and review animated shows they have watched. Users can rate cartoons, write short reviews, and organize their viewing history. The goal of the app is to provide a simple way to keep track of one's favorite animated series and contribute to a community that share a similar passion for animation.

##  The Inital Planned Features

- User authentication (signup and login)
- Add cartoon reviews with ratings
- Edit or delete reviews
- Filter cartoons by rating or watch status
- View a statistics dashboard personalized to each user

If things go well I will additionally add...
- Categories, tags, or advanced filtering
- Media Uploads
- User Profile Pages

## Tech Stack

- React
- CSS
- Supabase (Backend)
- Netlify (Deployment)
